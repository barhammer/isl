# ISL Hand Tracking Pipeline

## Setup

```bash
git clone https://github.com/barhammer/isl.git
cd isl
conda env create -f environment.yml
conda activate isl
```

## Run

```bash
python main.py
```

## Requirements

* Webcam
* Python 3.10
* Conda

## Notes

* Works best with good lighting
* Supports multiple people
